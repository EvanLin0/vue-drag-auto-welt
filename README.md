# vue-drag-auto-welt
>  基于vue的可拖动的悬浮球自动贴边

* 感谢github中 vencent2017 大佬提供的代码

## Build Setup

``` bash
基于 vue 的可拖动的悬浮球，自动贴边
支持属性：value 事件 @click 插槽 value
使用示例：<drag-ball :value="message" @click="alert('悬浮球事件')"><div slot="value">悬浮球！</div></drag-ball>
npm i vue-drag-ball
import dragBall from 'vue-drag-ball'
Vue.use(dragBall)
```
